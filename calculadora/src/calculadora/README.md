# Calculadora em Java (Console)

Este projeto é uma **calculadora simples em Java**, desenvolvida para praticar conceitos fundamentais da linguagem, como entrada de dados pelo terminal, métodos, controle de fluxo e tratamento de exceções.

## Funcionalidades
  Soma
  Subtração
  Multiplicação
  Divisão  
 Tratamento de erro para **divisão por zero**
 Validação básica de entradas inválidas

## Tecnologias Utilizadas
- Java
- IntelliJ IDEA
- `java.util.Scanner` para entrada de dados
- `Locale.US` para leitura correta de números decimais

## Estrutura do Projeto
calculadora/Calculator.java

## Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/java-basics.git
2. Abra o projeto no IntelliJ IDEA

3. Navegue até o pacote:
calculadora
4. Execute a classe Calculator

5. Insira:

Primeiro número

Operação desejada (+ - * /)

Segundo número

## Conceitos Praticados
Métodos estáticos

Estrutura switch

Tratamento de exceções (try / catch / finally)

Exceções personalizadas (ArithmeticException)

Boas práticas iniciais em Java

Entrada e saída pelo terminal
